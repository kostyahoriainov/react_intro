    При выборе пользователя с помощью соответствующего checkbox, имя только выбранного пользователя должно быть перечеркнутым (класс "checkbox"), и checkbox должен быть включенным только у выбранного пользователя.

    State, функция-обработчик должены быть в компоненте UserList и передаваться в дочерний комронент User через props
